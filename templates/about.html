Motivation

The goal of this project is to provide the user with an easy and fun way to create musically-tuned vocals from an untrained voice. Giving the user tools right at their fingertips like auto-tune profiles, EQ, and reverb settings allows for a fun and engaging tool to use. Using the Audealize API not only provides a good example of what it can do, it also provides the user with tested reverb and EQ profiles that will be what the user expects to hear. Finally, building easy-to-use creativity tools of this ilk is important. Industry-standard autotuning and sound manipulation software is designed for professionals, leaving a barrier to entry for the casual user. Our tool is designed for casual users who are just becoming interesting or could become interested in audio production.

Process

Songbird's frontend is built using Flask, HTML5, and JavaScript. First, the user chooses the effects they wish to apply to their voice. This includes autotuning, which we accomplished using our own implmentation of the YIN pitch-tracking algorithm and our own pitch-shifting tactic. The user is able to choose between three autotune settings, which range from tuning to every possible note on an equal-tempered A440 scale to very few notes on an "absurd" setting. The user can also choose EQ and reverb profiles which was implemented using the Audealize API. Next, using RecorderJS, Songbird captures the user's vocals when the user hits record. The user then hits stop when they are done recording. Once the processing on the backend is complete, the play button will become available to click and the user can hear the final result.
